## Step by Step

### Prerequisites

- Make sure to have nvm installed, and run `nvm install`, `nvm use` on the repo root.
- Install Tampermonkey

### Install dependencies

Run `npm i`.

### Build app

Run `npm run build`.

### Copy generated script into Tampermonkey

Copy the file contents of `dist/script.js` and paste it into Tampermonkey as a new script or update the existing one.
